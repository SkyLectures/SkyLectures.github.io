## 1. Office 365 기반의 간단한 Copilot 활용 예제

- **실습 준비물**
    - Microsoft 365 Copilot이 활성화된 Microsoft 365 계정 (필요 시 교육용 또는 평가판 계정 활용)
    - Microsoft Word 또는 PowerPoint, Outlook 애플리케이션

### 1.1 Word에서 보고서 초안 빠르게 만들기

1.  **Word 실행 및 새 문서 열기**:
    *   Microsoft Word를 실행하고 '새 문서'를 클릭합니다.
2.  **Copilot 호출**:
    *   문서 화면 중앙에 나타나는 'Draft with Copilot' (Copilot으로 초안 작성) 프롬프트 창을 클릭하거나, 리본 메뉴의 'Copilot' 버튼을 클릭합니다.
3.  **프롬프트 입력**:
    *   다음과 같이 입력하고 Enter를 누릅니다:
        ```
        새로운 스마트 오피스 도입을 위한 500자 분량의 제안서 초안을 작성해 줘. 제안서에는 스마트 오피스의 정의, 기대 효과, 도입 방안이 포함되어야 해.
        ```
4.  **결과 확인 및 편집**:
    *   Copilot이 생성한 초안을 확인합니다.
    *   필요에 따라 초안을 수정하거나, "더 상세하게 작성해 줘", "도입 방안을 좀 더 구체적으로 설명해 줘" 등 추가 프롬프트를 통해 내용을 다듬을 수 있습니다.

### 1.2 PowerPoint에서 아이디어 프레젠테이션 구성

1.  **PowerPoint 실행 및 새 프레젠테이션 열기**:
    *   Microsoft PowerPoint를 실행하고 '새 프레젠테이션'을 클릭합니다.
2.  **Copilot 호출**:
    *   새 슬라이드에 나타나는 'Draft with Copilot' (Copilot으로 초안 작성) 프롬프트 창을 클릭하거나, 리본 메뉴의 'Copilot' 버튼을 클릭합니다.
3.  **프롬프트 입력**:
    *   다음과 같이 입력하고 Enter를 누릅니다:
        ```
        다가오는 기술 혁신 트렌드에 대한 3장짜리 프레젠테이션 초안을 작성해 줘. 주요 키워드는 AI, 사물 인터넷(IoT), 친환경 기술이야.
        ```
4.  **결과 확인 및 슬라이드 생성**:
    *   Copilot이 제시하는 프레젠테이션 아웃라인을 확인합니다.
    *   만족스럽다면 'Create Presentation' (프레젠테이션 생성) 버튼을 클릭하여 슬라이드를 만듭니다.
    *   각 슬라이드 내용을 수정하거나 추가적인 이미지, 데이터 등을 Copilot에게 요청할 수 있습니다.

### 1.3 Outlook에서 이메일 요약 및 답장 초안 작성

- 시간에 여유가 있으면 실행

1.  **Outlook 실행 및 긴 이메일 열기**:
    *   Microsoft Outlook을 실행하고 내용이 긴 이메일 스레드를 하나 선택합니다.
2.  **Copilot 호출**:
    *   이메일 본문 위에 나타나는 'Summarize with Copilot' (Copilot으로 요약) 버튼을 클릭합니다.
3.  **요약 확인**:
    *   Copilot이 이메일의 핵심 내용을 요약해 주는 것을 확인합니다.
4.  **답장 초안 작성**:
    *   'Reply with Copilot' (Copilot으로 답장) 버튼을 클릭합니다.
    *   프롬프트 창에 "이메일 내용에 대해 긍정적으로 동의하고, 다음 단계에 대한 구체적인 일정을 제안하는 답장을 작성해 줘." 와 같이 입력하여 초안을 생성하고 수정합니다.
