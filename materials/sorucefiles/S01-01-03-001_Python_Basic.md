# 3.1 프로그래밍이란?

- 컴퓨터에게 일을 시키기 위한 명령어 모음을 만드는 일<br/>
→ 주어진 데이터를 우리가 원하는 형태로 가공, 처리하는 과정을 정리한 것
- 프로그래밍의 단계: 데이터 입력 → 데이터 처리 → 데이터 출력
